#Брендинг

Лоппи

Разработка бренда серии медовых батончиков

В рамках проекта по продуктовому брендингу для производителя тортов Есаян мы создали серию этикеток для медовых ломтиков для детей. Мы придумали четырёх персонажей, которые стали лицом этой линейки, и назвали их Лоппиками. Каждый Лоппик представляет свой вкус медового батончика. И у каждого Лоппика есть свои атрибуты.
